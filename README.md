## Overview

VitaMind is an ultra-modern, professional landing page for an AI-powered mental health companion. Designed with a strong, credible, and inspiring visual identity, VitaMind leverages advanced UX/UI trends and a unique color palette to deliver a seamless, trustworthy, and captivating user experience.

- **Palette:**  
  - Deep Blue-Green: `#518591` (primary, accents, buttons)  
  - Digital Gold: `#e3b01c` (premium details, strategic CTAs, icons)  
  - Pure White: `#ffffff` (backgrounds, clarity, readability)

- **Key Features:**  
  - Responsive, accessible, and mobile-first design  
  - Futuristic, dynamic animations and glassmorphism effects  
  - English-only, professional copywriting  
  - Modular React/Next.js components  
  - Tailwind CSS for rapid, consistent styling  
  - Fully open-source and ready for deployment

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [pnpm](https://pnpm.io/) or [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
git clone https://github.com/chedi-ouerghi/VitaminD.git
cd VitaminD
pnpm install
# or
npm install
# or
yarn install
```

### Development

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

### Production Build

```bash
pnpm build
pnpm start
# or
npm run build && npm start
# or
yarn build && yarn start
```
